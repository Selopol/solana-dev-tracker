# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Design Guidelines

The icons should represent the Solana Developer Tracker brand:
- Use the purple gradient theme (#667eea to #764ba2)
- Include Solana-related visual elements (e.g., stylized "S" or blockchain symbol)
- Ensure icons are clear and recognizable at all sizes

## Generating Icons

You can use any image editing software to create these icons, or use online tools like:
- Figma
- Adobe Illustrator
- Canva
- Online PNG generators

Make sure to export in PNG format with transparency.
